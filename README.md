# School Management - Next.js Demo

This is a demo Next.js (app router) project generated by ChatGPT. It uses Tailwind + Firebase (optional) + Chart.js.

Run locally:
1. copy `.env.local.example` to `.env.local` and fill Firebase keys (optional)
2. npm install
3. npm run dev

Notes: This project uses an in-memory store by default for demo purposes.
